# Tmux 快捷键列表 (Oh my tmux!)

> 基于您的 `.tmux.conf` 和 `.tmux.conf.local` 配置生成的完整快捷键备忘录。

**前缀键 (Prefix)**: `Ctrl + b` 或 `Ctrl + a` (以下用 `<Prefix>` 表示)

## 🔌 常用插件快捷键

### 会话保存与恢复 (tmux-resurrect)
| 快捷键 | 功能 |
| :--- | :--- |
| `<Prefix> + Ctrl + s` | **保存** 当前所有会话状态 |
| `<Prefix> + Ctrl + r` | **恢复** 之前保存的会话状态 |

### 快速打开文件/链接 (tmux-open)
*(仅在复制模式下有效)*
| 快捷键 | 功能 |
| :--- | :--- |
| `o` | **打开** 选中的文件或链接 (使用系统默认程序) |
| `Ctrl + o` | **编辑** 选中的文件 (使用 `$EDITOR`) |
| `Shift + s` | **搜索** 选中的内容 (使用 Google) |

### 快捷键查询
| 快捷键 | 功能 |
| :--- | :--- |
| `<Prefix> + i` | **打开本列表** (分屏显示) |

---

## 🛠 通用操作 (General)
| 快捷键 | 功能 |
| :--- | :--- |
| `<Prefix> + e` | **编辑** 配置文件 (`.tmux.conf.local`) |
| `<Prefix> + r` | **重载** 配置文件 (使修改生效) |
| `<Prefix> + m` | 切换 **鼠标** 模式 (开启/关闭) |
| `Ctrl + l` | **清除屏幕** (无需前缀，类似 `clear` 命令) |
| `<Prefix> + ?` | **列出所有快捷键** (官方帮助) |
| `<Prefix> + :` | **进入命令行模式** |
| `<Prefix> + t` | **显示时钟** |
| `<Prefix> + d` | **分离** (Detach) 当前会话 |

## 🪟 窗口与面板 (Windows & Panes)
| 快捷键 | 功能 |
| :--- | :--- |
| `<Prefix> + -` | **水平分割** 窗口 (上下分屏) |
| `<Prefix> + _` | **垂直分割** 窗口 (左右分屏) |
| `<Prefix> + x` | **关闭面板** (取消分屏，需按 y 确认) |
| `<Prefix> + &` | **关闭窗口** (需按 y 确认) |
| `<Prefix> + h / j / k / l` | **移动光标** (左 / 下 / 上 / 右) |
| `<Prefix> + H / J / K / L` | **调整大小** (向 左 / 下 / 上 / 右) |
| `<Prefix> + +` | **最大化** 当前面板 (再按一次恢复) |
| `<Prefix> + >` | 与 **下一个** 面板交换位置 |
| `<Prefix> + <` | 与 **上一个** 面板交换位置 |
| `<Prefix> + Ctrl + h` | 切换到 **上一个窗口** |
| `<Prefix> + Ctrl + l` | 切换到 **下一个窗口** |
| `<Prefix> + Tab` | 切换到 **最近使用的窗口** |
| `<Prefix> + c` | **新建窗口** |
| `<Prefix> + ,` | **重命名窗口** |
| `<Prefix> + w` | **选择窗口** (交互式列表) |
| `<Prefix> + q` | **显示面板编号** (按数字跳转) |
| `<Prefix> + Ctrl + Shift + h` | **交换窗口** (与前一个窗口交换) |
| `<Prefix> + Ctrl + Shift + l` | **交换窗口** (与后一个窗口交换) |

## 🚀 会话管理 (Sessions)
| 快捷键 | 功能 |
| :--- | :--- |
| `<Prefix> + s` | **打开会话列表** (在此界面按 `x` 删除选中会话) |
| `<Prefix> + Ctrl + c` | **新建** 会话 |
| `<Prefix> + Ctrl + f` | **查找** 会话 (按名称跳转) |
| `<Prefix> + Shift + Tab` | 切换到 **上一个会话** |
| `<Prefix> + (` | 切换到 **上一个会话** |
| `<Prefix> + )` | 切换到 **下一个会话** |
| `<Prefix> + $` | **重命名** 当前会话 |

## 📋 复制与粘贴 (Copy & Paste)
| 快捷键 | 功能 |
| :--- | :--- |
| `<Prefix> + Enter` | **进入复制模式** |
| `<Prefix> + b` | **列出剪贴板历史** (选择并粘贴) |
| `<Prefix> + p` | **粘贴** (最新的一条) |
| `<Prefix> + P` (大写) | **选择粘贴** (选择历史记录) |

**在复制模式中 (Vi 风格):**
*   `v`: 开始选择 (Visual mode)
*   `Ctrl + v`: 矩形块选择
*   `y`: 复制选中文本并退出 (Yank)
*   `Esc`: 退出复制模式
*   `H` / `L`: 跳转到 行首 / 行尾

## 🔗 高级工具集成 (如果已安装)
| 快捷键 | 功能 |
| :--- | :--- |
| `<Prefix> + F` | **Facebook PathPicker** (选择屏幕上的文件路径) |
| `<Prefix> + U` | **Urlscan / Urlview** (列出并打开 URL) |

## 🔌 插件管理 (TPM)
| 快捷键 | 功能 |
| :--- | :--- |
| `<Prefix> + I` (大写) | **安装** 新添加的插件 |
| `<Prefix> + u` | **更新** 插件 |
| `<Prefix> + Alt + u` | **卸载** 已移除的插件 |
