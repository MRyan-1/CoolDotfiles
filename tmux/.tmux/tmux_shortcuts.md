# Tmux 快捷键列表 (Oh my tmux!)

> 基于您的 `.tmux.conf` 和 `.tmux.conf.local` 配置生成的完整快捷键备忘录。

**前缀键 (Prefix)**: `Ctrl + b` 或 `Ctrl + a` (以下用 `<Prefix>` 表示)

## 🔌 常用插件快捷键


### 快捷键查询
| 快捷键 | 功能 |
| :--- | :--- |
| `<Prefix> + i` | **打开本列表** (分屏显示) |

---

## 🛠 通用操作 (General)
| 快捷键 | 功能 |
| :--- | :--- |
| `<Prefix> + e` | **编辑** 配置文件 (`.tmux.conf.local`) |
| `<Prefix> + r` | **重载** 配置文件 (使修改生效) |
| `Ctrl + l` | **清除屏幕** (无需前缀，类似 `clear` 命令) |
| `<Prefix> + d` | **分离** (Detach) 当前会话 |

## 🪟 窗口与面板 (Windows & Panes)
| 快捷键 | 功能 |
| :--- | :--- |
| `<Prefix> + -` | **水平分割** 窗口 (上下分屏) |
| `<Prefix> + _` | **垂直分割** 窗口 (左右分屏) |
| `<Prefix> + x` | **关闭面板** (取消分屏，需按 y 确认) |
| `<Prefix> + &` | **关闭窗口** (需按 y 确认) |
| `<Prefix> + h / j / k / l` | **移动光标** (左 / 下 / 上 / 右) |
| `<Prefix> + H / J / K / L` | **调整大小** (向 左 / 下 / 上 / 右, 每次 5 格) |
| `⌥+⇧+k` | **增高面板** (无需前缀, 每次 1 格) |
| `⌥+⇧+j` | **减高面板** (无需前缀, 每次 1 格) |
| `⌥+⇧+l` | **增宽面板** (无需前缀, 每次 1 格) |
| `⌥+⇧+h` | **减宽面板** (无需前缀, 每次 1 格) |
| `<Prefix> + +` | **最大化** 当前面板 (再按一次恢复) |
| `<Prefix> + >` | 与 **下一个** 面板交换位置 |
| `<Prefix> + <` | 与 **上一个** 面板交换位置 |
| `<Prefix> + Ctrl + h` | 切换到 **上一个窗口** |
| `<Prefix> + Ctrl + l` | 切换到 **下一个窗口** |
| `<Prefix> + Tab` | 切换到 **最近使用的窗口** |
| `<Prefix> + c` | **新建窗口** |
| `<Prefix> + ,` | **重命名窗口** |
| `<Prefix> + w` | **选择窗口** (交互式列表) |
| `<Prefix> + q` | **显示面板编号** (按数字跳转) |
| `<Prefix> + Ctrl + Shift + h` | **交换窗口** (与前一个窗口交换) |
| `<Prefix> + Ctrl + Shift + l` | **交换窗口** (与后一个窗口交换) |

## 🚀 会话管理 (Sessions)
| 快捷键 | 功能 |
| :--- | :--- |
| `<Prefix> + s` | **打开会话列表** (在此界面按 `x` 删除选中会话) |
| `<Prefix> + Ctrl + c` | **新建** 会话 |
| `<Prefix> + Ctrl + f` | **查找** 会话 (按名称跳转) |
| `<Prefix> + Shift + Tab` | 切换到 **上一个会话** |
| `<Prefix> + (` | 切换到 **上一个会话** |
| `<Prefix> + )` | 切换到 **下一个会话** |
| `<Prefix> + $` | **重命名** 当前会话 |
