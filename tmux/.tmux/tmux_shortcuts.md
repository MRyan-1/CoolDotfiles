# Tmux 快捷键列表 (Oh my tmux!)

> 基于您的 `.tmux.conf` 和 `.tmux.conf.local` 配置生成的完整快捷键备忘录。

**前缀键 (Prefix)**: `Ctrl + b` 或 `Ctrl + a` (以下用 `<Prefix>` 表示)

## 🔌 常用插件快捷键

### 快捷键查询
| 快捷键 | 功能 |
| :--- | :--- |
| `<Prefix> + i` | **打开本列表** (分屏显示) |

---

## 🛠 通用操作 (General)
| 快捷键 | 功能 |
| :--- | :--- |
| `<Prefix> + e` | **编辑** 配置文件 (`.tmux.conf.local`) |
| `<Prefix> + r` | **重载** 配置文件 (使修改生效) |
| `Ctrl + l` | **清除屏幕** (无需前缀，类似 `clear` 命令) |
| `<Prefix> + d` | **分离** (Detach) 当前会话 |
| `<Prefix> + m` | **切换鼠标模式** (开启/关闭) |
| `<Prefix> + t` | **显示时钟** |
| `<Prefix> + :` | **命令行模式** (输入 tmux 命令) |
| `<Prefix> + ?` | **显示所有快捷键** (官方列表) |

## 🪟 窗口与面板 (Windows & Panes)
| 快捷键 | 功能 |
| :--- | :--- |
| `<Prefix> + -` | **水平分割** 窗口 (上下分屏) |
| `<Prefix> + _` | **垂直分割** 窗口 (左右分屏) |
| `<Prefix> + x` | **关闭面板** (取消分屏，需按 y 确认) |
| `<Prefix> + &` | **关闭窗口** (需按 y 确认) |
| `<Prefix> + h / j / k / l` | **移动光标** (左 / 下 / 上 / 右) |
| `<Prefix> + H / J / K / L` | **调整大小** (向 左 / 下 / 上 / 右, 每次 5 格) |
| `<Prefix> + +` | **最大化** 当前面板 (再按一次恢复) |
| `<Prefix> + >` | 与 **下一个** 面板交换位置 |
| `<Prefix> + <` | 与 **上一个** 面板交换位置 |
| `<Prefix> + Ctrl + h` | 切换到 **上一个窗口** |
| `<Prefix> + Ctrl + l` | 切换到 **下一个窗口** |
| `<Prefix> + Tab` | 切换到 **最近使用的窗口** |
| `<Prefix> + c` | **新建窗口** |
| `<Prefix> + ,` | **重命名窗口** |
| `<Prefix> + w` | **选择窗口** (交互式列表) |
| `<Prefix> + q` | **显示面板编号** (按数字跳转) |
| `<Prefix> + Ctrl + Shift + h` | **交换窗口** (与前一个窗口交换) |
| `<Prefix> + Ctrl + Shift + l` | **交换窗口** (与后一个窗口交换) |
| `<Prefix> + !` | **将当前面板拆分为独立窗口** |
| `<Prefix> + {` | **向前移动面板** |
| `<Prefix> + }` | **向后移动面板** |
| `<Prefix> + o` | **循环切换面板** |
| `<Prefix> + z` | **最大化/还原当前面板** (Zoom) |

## 🚀 会话管理 (Sessions)
| 快捷键 | 功能 |
| :--- | :--- |
| `<Prefix> + s` | **打开会话列表** (在此界面按 `x` 删除选中会话) |
| `<Prefix> + Ctrl + c` | **新建** 会话 |
| `<Prefix> + Ctrl + f` | **查找** 会话 (按名称跳转) |
| `<Prefix> + Shift + Tab` | 切换到 **上一个会话** |
| `<Prefix> + (` | 切换到 **上一个会话** |
| `<Prefix> + )` | 切换到 **下一个会话** |
| `<Prefix> + $` | **重命名** 当前会话 |
| `<Prefix> + L` | **切换到最后一个会话** |

## 📋 复制模式 (Copy Mode)
| 快捷键 | 功能 |
| :--- | :--- |
| `<Prefix> + [` | **进入复制模式** (使用 vi 键位) |
| `<Prefix> + ]` | **粘贴** |
| `<Prefix> + =` | **选择粘贴缓冲区** |
| **复制模式下** | |
| `v` | **开始选择** (Visual selection) |
| `y` | **复制选中内容** (Yank) |
| `Esc` | **退出复制模式** |
| `Space` | **开始选择** |
| `Enter` | **复制并退出** |
| `/` | **向下搜索** |
| `?` | **向上搜索** |
| `n` | **下一个搜索结果** |
| `N` | **上一个搜索结果** |

## 🧩 插件管理 (TPM)
| 快捷键 | 功能 |
| :--- | :--- |
| `<Prefix> + I` | **安装** 新插件 |
| `<Prefix> + u` | **更新** 插件 |
| `<Prefix> + Alt + u` | **卸载** 未使用的插件 |

## 🔌 高级插件功能 (Resurrect & Continuum)
| 快捷键 | 功能 |
| :--- | :--- |
| `<Prefix> + Ctrl + s` | **保存** 当前环境 (Save) |
| `<Prefix> + Ctrl + r` | **恢复** 环境 (Restore) |
