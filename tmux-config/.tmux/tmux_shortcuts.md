# Tmux 快捷键列表 (Oh my tmux!)

> 基于您的 `.tmux.conf` 和 `.tmux.conf.local` 配置生成的完整快捷键备忘录。

**前缀键 (Prefix)**: `Ctrl + b` 或 `Ctrl + a` (以下用 `<Prefix>` 表示)

## 🔌 常用插件快捷键

### 快捷键查询

| 快捷键         | 功能                            |
| :------------- | :------------------------------ |
| `<Prefix> + ?` | **打开本列表** (弹窗显示，推荐) |

---

## 🛠 通用操作 (General)

| 快捷键         | 功能                                       |
| :------------- | :----------------------------------------- |
| `<Prefix> + e` | **编辑** 配置文件 (`.tmux.conf.local`)     |
| `<Prefix> + r` | **重载** 配置文件 (使修改生效)             |
| `Ctrl + l`     | **清除屏幕** (无需前缀，类似 `clear` 命令) |
| `<Prefix> + d` | **分离** (Detach) 当前会话                 |
| `<Prefix> + i` | **分屏查看** 快捷键手册 (右侧 40%)         |
| `<Prefix> + m` | **切换鼠标模式** (开启/关闭)               |
| `<Prefix> + :` | **命令行模式** (输入 tmux 命令)            |

## 🪟 窗口与面板 (Windows & Panes)

| 快捷键                     | 功能                                           |
| :------------------------- | :--------------------------------------------- |
| `<Prefix> + v`             | **竖分** 窗格 (左右分屏, vim 风格)             |
| `<Prefix> + s`             | **横分** 窗格 (上下分屏, vim 风格)             |
| `<Prefix> + x`             | **关闭面板** (直接关闭，无需确认)              |
| `<Prefix> + X`             | **关闭窗口** (直接关闭)                        |
| `<Prefix> + S`             | **同步输入** 开关 (所有窗格同时输入)           |
| `<Prefix> + H / J / K / L` | **调整大小** (向 左 / 下 / 上 / 右, 每次 2 格) |
| `<Prefix> + +`             | **最大化** 当前面板 (再按一次恢复)             |
| `<Prefix> + <`             | **窗口左移** (与前一个窗口交换位置)            |
| `<Prefix> + >`             | **窗口右移** (与后一个窗口交换位置)            |
| `<Prefix> + Ctrl + h`      | 切换到 **上一个窗口**                          |
| `<Prefix> + Ctrl + l`      | 切换到 **下一个窗口**                          |
| `<Prefix> + Tab`           | 切换到 **最近使用的窗口**                      |
| `<Prefix> + c`             | **新建窗口**                                   |
| `<Prefix> + ,`             | **重命名窗口**                                 |
| `<Prefix> + w`             | **选择窗口** (交互式列表)                      |
| `<Prefix> + q`             | **显示面板编号** (按数字跳转)                  |
| `<Prefix> + !`             | **将当前面板拆分为独立窗口**                   |
| `<Prefix> + {`             | **向前移动面板**                               |
| `<Prefix> + }`             | **向后移动面板**                               |
| `<Prefix> + z`             | **最大化/还原当前面板** (Zoom)                 |

## 🚀 会话管理 (Sessions)

| 快捷键                   | 功能                                           |
| :----------------------- | :--------------------------------------------- |
| `<Prefix> + o`           | **打开 sessionx** (会话/窗口切换器，带预览)    |
| `<Prefix> + w`           | **打开会话列表** (在此界面按 `x` 删除选中会话) |
| `<Prefix> + Ctrl + c`    | **新建** 会话                                  |
| `<Prefix> + Ctrl + f`    | **查找** 会话 (按名称跳转)                     |
| `<Prefix> + Shift + Tab` | 切换到 **上一个会话**                          |
| `<Prefix> + (`           | 切换到 **上一个会话**                          |
| `<Prefix> + )`           | 切换到 **下一个会话**                          |
| `<Prefix> + $`           | **重命名** 当前会话                            |
| `<Prefix> + L`           | **切换到最后一个会话**                         |

## 📋 复制模式 (Copy Mode)

| 快捷键         | 功能                            |
| :------------- | :------------------------------ |
| `<Prefix> + Enter` | **进入复制模式** (使用 vi 键位)    |
| `<Prefix> + [`     | **进入复制模式** (同上，别名)      |
| `<Prefix> + b`     | **列出** 粘贴缓冲区               |
| `<Prefix> + p`     | **粘贴** (从最新缓冲区)           |
| `<Prefix> + P`     | **选择** 粘贴缓冲区               |
| **复制模式下** |                                 |
| `v`            | **开始选择** (Visual selection) |
| `y`            | **复制选中内容** (Yank)         |
| `Esc`          | **退出复制模式**                |
| `Space`        | **开始选择**                    |
| `Enter`        | **复制并退出**                  |
| `/`            | **向下搜索**                    |
| `?`            | **向上搜索**                    |
| `n`            | **下一个搜索结果**              |
| `N`            | **上一个搜索结果**              |
| **鼠标滚轮**   | **半页滚动** (上/下)            |

## 🖱 鼠标操作 (Mouse)

| 操作                 | 功能                             |
| :------------------- | :------------------------------- |
| 点击左侧状态栏       | 打开**会话树** (choose-tree)     |
| 点击窗口标签         | **切换窗口**                     |
| 鼠标滚轮 (普通模式)  | 自动进入复制模式并滚动           |

## 🔍 弹窗工具 (Popups)

| 快捷键         | 功能                                           |
| :------------- | :--------------------------------------------- |
| `<Prefix> + f` | **弹窗** 打开子 shell (当前目录)               |
| `<Prefix> + g` | **弹窗** 打开 lazygit (当前目录)               |
| `<Prefix> + T` | **弹窗** 打开 htop                             |
| `<Prefix> + /` | **弹窗** fzf 文件查找 (当前目录)               |
| `<Prefix> + ?` | **弹窗** 快捷键速查                            |
| `<Prefix> + N` | **弹窗** 笔记本 ($HOME/notes/<目录名>-note.md) |

## 🔗 Vim-Tmux 导航 (vim-tmux-navigator)

| 快捷键     | 功能                             |
| :--------- | :------------------------------- |
| `Ctrl + h` | 向左切换窗格 (tmux/vim 无缝)     |
| `Ctrl + j` | 向下切换窗格 (tmux/vim 无缝)     |
| `Ctrl + k` | 向上切换窗格 (tmux/vim 无缝)     |
| `Ctrl + l` | 向右切换窗格 (tmux/vim 无缝)     |
| `Ctrl + \` | 切换到上一个窗格 (tmux/vim 无缝) |

## 🔎 tmux-fzf

| 快捷键         | 功能                                               |
| :------------- | :------------------------------------------------- |
| `<Prefix> + F` | **tmux-fzf** 模糊搜索 (session/window/pane/命令等) |

## 🔄 tmux-sessionx (Session 切换器)

| 快捷键         | 功能                                                   |
| :------------- | :----------------------------------------------------- |
| `<Prefix> + o` | **打开 sessionx** (带实时预览的 session/window 切换器) |
|                | 内部快捷键:                                            |
| `Ctrl + t`     | 切换到树形模式                                         |
| `Ctrl + w`     | 切换到窗口模式                                         |
| `Ctrl + x`     | 切换到配置目录浏览                                     |
| `Ctrl + r`     | 重命名 session                                         |
| `Ctrl + e`     | 从当前目录新建 session                                 |
| `Ctrl + f`     | 从 zoxide 路径新建 session                             |
| `Alt + Bksp`   | 删除选中 session                                       |
| `?`            | 切换预览开/关                                          |

## 👆 tmux-thumbs (快速复制)

| 快捷键             | 功能                                                   |
| :----------------- | :----------------------------------------------------- |
| `<Prefix> + Space` | 激活 thumbs 模式，屏幕上的 URL/路径/哈希等出现字母提示 |
|                    | 按对应字母即可复制到剪贴板                             |

## 🧩 插件管理 (TPM)

| 快捷键               | 功能                  |
| :------------------- | :-------------------- |
| `<Prefix> + I`       | **安装** 新插件       |
| `<Prefix> + u`       | **更新** 插件         |
| `<Prefix> + Alt + u` | **卸载** 未使用的插件 |

## 🔌 高级插件功能 (Resurrect & Continuum)

| 快捷键                | 功能                     |
| :-------------------- | :----------------------- |
| `<Prefix> + Ctrl + s` | **保存** 当前环境 (Save) |
| `<Prefix> + Ctrl + r` | **恢复** 环境 (Restore)  |
